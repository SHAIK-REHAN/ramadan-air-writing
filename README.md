# Ramadan Air Writing ✨🌙

A computer vision project that lets you **write in the air using hand gestures**.

This project uses **MediaPipe Hands** and **HTML Canvas** to track hand landmarks and draw neon strokes following your finger.

---

## 🚀 Features
- Real-time hand tracking using MediaPipe
- Neon air-writing effect
- Webcam-based gesture interaction
- Smooth drawing using canvas

---

## 🛠️ Technologies Used
- JavaScript
- HTML5 Canvas
- MediaPipe Hands
- Web Camera API

---

## ▶️ How to Run

1. Clone the repository

2. Open the folder

3. Run using **Live Server** in VS Code

4. Allow camera access

---

## ✋ Controls

- Hold **SHIFT key** → Start drawing
- Release **SHIFT key** → Move to next letter
- Press **SPACEBAR** → Clear screen

---
